# 🌤 Weather App (SwiftUI)

A modern and minimal **Weather application built using SwiftUI**, leveraging **async/await** for networking and real-time weather data from the WeatherAPI.  
The app features a clean UI, dark/light mode support, and smooth user experience inspired by modern iOS design.

---

## ✨ Features

- 🔍 Search weather by city name
- 🌡 Real-time temperature display
- 🤒 “Feels like” temperature
- 🌤 Weather condition icon + description
- 📍 Location-based weather details
- 🌙 Dark / Light mode support
- ⏳ Loading indicator during API calls
- ❌ Error handling for invalid input and network issues
- 🎨 Modern glassmorphism-style UI
